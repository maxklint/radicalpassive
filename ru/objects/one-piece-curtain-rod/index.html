Монолитный карниз для штор

Обычно карнизы для штор состоят из нескольких частей: штанга, кронштейны, которые держат штангу, и наконечники по краям штанги. Штанга может быть телескопической, кронштейн может быть раздвижным, а наконечники - в форме ракушек. Это конечно же функционально, но с другой стороны стандартизированные размеры и массовое изготовление этого мебельного элемента свели его эстетическую ценность на нет. В противовес этому, мы сделали карниз без лишних деталей: единая сварная металлическая конструкция, покрытая чёрной краской. Карниз, который был сделан для определённого интерьера, по определённым размерам, как настоящий дизайнерский объект. Разработан для Melancholy space.
